# oop-laba1

В лабораторной работе используется фреймворк Qt.

Компилировалось всё по Linux. 


## Как сбилдить

Под Windows легче всего будет открыть проект в VisualStudio.

Под Linux:
```
$ git clone https://github.com/den-a-s/oop-laba1
$ mkdir build && cd build
$ cmake ../os-asmt-passwordgen
$ make
$ ./laba1
```
